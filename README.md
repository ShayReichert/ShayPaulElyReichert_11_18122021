**Lancer l'application en local** :

Cloner le projet :
```
$ git clone https://github.com/ShayReichert/ShayPaulElyReichert_11_18122021
```

Aller au repo cloné :
```
$ cd ShayPaulElyReichert_11_18122021
```

Installez les packages npm :
```
$ npm install
```

Lancer l'application React :
```
$ npm start
```
